Set salesorders to invoiced

Correct the automatic workflow in Odoo when salesorders don't go to invoiced

This module gives the user the option to correct the invoice state of the salesorders within Odoo. Normally a salesorder will go to the invoiced state when all salesorderlines are delivered and invoiced. Sometimes there are differencies in amount and quantities in the invoice. This will keep the salesorder in the state to_invoice. With the help of this module you can update the invoice_state.

In this way the salesorders with the state to_invoice will kept clean and companies can focus on invoicing the salesorders within this state

Setting the salesorder to invoiced

  1. Select the salesorder which has the state to_invoice
  2. Press the action button
  3. Choose hte menu option "set salesorder to invoiced"
  4. Provide a reason for the manual update of the status and press the button "dosettoinvoiced"
